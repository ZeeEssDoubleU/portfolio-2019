A Youtube skeleton clone built with React, Redux and the Youtube API. Uses Redux Saga middleware to help manage Redux state. Implements features such as lazy loading results, search functionality, trending videos, liked and saved videos. Built for learning purposes and personal use.

Github Issue Tracker
App that tracks repository issues on Github. Built for learning purposes with Apollo and React. This project served as my introduction to GraphQL and Apollo. Features include search functionality, optimistic UI updates, server-side result filtering, like/unlike and commenting.
